IPANEMA SUMMER HITS COUTDOWN

SUMMER HITS COUNTDOWN es una aplicación sencilla, simplemente el usuario tiene que ingresar la fecha de sus próximas vacaciones. A partir de ese momento comienza una cuenta regresiva y se desbloquea una playlist con hits de veranos pasados que se iran incrementado con el correr de los dias.

NECESIDAD DEL NEGOCIO

Realizar una aplicación sencilla para promocionar las nuevas sandalias Ipanema.

TECNOLOGIAS UTILIZADAS Y PLATAFORMAS

La app para IOS fue desarrollada en xcode 6.1, Objective C. La app para Androd fue desarrollada con Android Studio 1.2, Java.
